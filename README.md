# Student Grade Management System

## Description
This is a simple Pyhton project that stores student marks, 
calculates the average score, and assigns grades.

## Features
- Stores student marks
- Calculate average score
- Assigns grade automatically
- Display student report

## Technologies Used
Python

## How to Run

1. Install Python
2. Download the project
3. Run the program

python grade_system.py
